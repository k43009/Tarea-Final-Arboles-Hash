## Proyecto Final: Árboles Binarios y Hash

- Binary Search Tree (BST): inserción, recorrido in-order, pre-order y post-order (recursivo e iterativo).
- HashTableChaining: tabla hash con resolución de colisiones mediante listas ligadas.
- HashSet: basado en HashTableChaining, sin permitir elementos repetidos.
- LinkedList y SentinelLinkedList: manejo de nodos, inserciones y eliminaciones.
- DynamicArray y Stack: ejemplos de estructuras dinámicas con memoria automática.
- Figuras: clases polimórficas (Círculo, Cuadrado, Cubo, FiguraNLados, etc.) para practicar herencia y virtualización.

## Cómo compilar

Si estás usando VS Code con g++:

g++ -std=c++17 -Wall -Wextra -O0 -g *.cpp -o programa.exe
./programa.exe
